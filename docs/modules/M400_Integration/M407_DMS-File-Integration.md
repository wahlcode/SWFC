\# M407 – DMS / File Integration



\## Zweck

M407 bildet die Integration zu externen Dokumentenmanagementsystemen ab.



M407 beantwortet:



> \*\*Wie werden externe Dokumentensysteme eingebunden?\*\*



\---



\## Inhalt



\- Verbindung zu DMS

\- Dateiablage extern

\- Referenzierung von Dokumenten



\---



\## Verknüpfung



\- M104 – Documents



\---



\## Beispiele



\- große Dateien extern speichern

\- Dokumente verlinken statt speichern



\---



\## Regeln



\- SWFC bleibt Referenzsystem

\- DMS bleibt Speicher



\---



\## Kernsatz



> \*\*M407 = externe Dokumentenwelt anbinden\*\*
