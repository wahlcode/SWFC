# M804 – DevSecOps

---

## 🎯 Zweck

M804 definiert Sicherheitsanforderungen für Entwicklung und Deployment.

---

## Inhalt

- sichere Build-Prozesse
- sichere Deployment-Prozesse
- Sicherheitsprüfungen in Pipelines

---

## Regeln

- keine unsignierten Builds
- keine ungeprüften Deployments
- Sicherheitschecks vor Release

---

## Abgrenzung

- kein Runtime-Monitoring → M808
- kein Setup → M107
