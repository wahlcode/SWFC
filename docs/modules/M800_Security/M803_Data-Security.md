# M803 – Data Security

---

## 🎯 Zweck

M803 schützt Daten im System.

---

## Inhalt

- Schutz der Vertraulichkeit
- Schutz der Integrität
- Schutz vor unberechtigtem Zugriff

---

## Regeln

- sensible Daten müssen geschützt gespeichert werden
- Zugriff nur über geprüfte Berechtigungen
- keine direkte Datenmanipulation ohne Audit

---

## Mindestanforderungen

- Zugriffskontrolle über M806
- Auditpflicht über M805
- keine unkontrollierten Datenexports

---

## Abgrenzung

- keine Zugriffsauswertung → M806
- keine Audit-Speicherung → M805

---

## Status

- verbindlich
