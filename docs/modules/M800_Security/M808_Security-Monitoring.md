# M808 – Security Monitoring

---

## 🎯 Zweck

M808 erkennt sicherheitskritische Ereignisse.

---

## Inhalt

- Anomalien
- verdächtige Zugriffe
- ungewöhnliches Verhalten

---

## Regeln

- sicherheitsrelevante Muster müssen erkannt werden
- Alarme müssen ausgelöst werden
- Ereignisse müssen nachvollziehbar sein

---

## Abgrenzung

- keine Audit-Speicherung → M805
- keine Zugriffsauswertung → M806

---

## Status

- verpflichtend
